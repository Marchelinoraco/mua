# Product

Shadcn Admin is a responsive admin dashboard UI template built with React and shadcn/ui. It serves as a starting point and reference implementation for building admin panels and internal tools.

## Key Capabilities

- Authentication flows (sign in, sign up, OTP, forgot password)
- Dashboard with charts and metrics
- Feature modules: users, tasks, apps, chats, and settings
- Light/dark theming, RTL/LTR direction support, and configurable fonts
- Reusable data tables with sorting, filtering, pagination, and URL-synced state
- Global command menu, error pages (401/403/404/500/503), and "coming soon" placeholders

## Audience

Developers who need a polished, accessible, and extensible admin UI foundation. The project favors clean conventions and consistent patterns over feature completeness, so it can be cloned and extended for real applications.
